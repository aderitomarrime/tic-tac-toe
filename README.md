## Overview

A web-based Tic-Tac-Toe game that allows two players to compete on the same device in an interactive and intuitive way. Players take turns marking the board with X and O, and the game automatically detects wins or draws. The application dynamically updates the interface, ensuring a smooth and responsive user experience.


## Live Demo

Visit the website: [Live Demo](https://aderitomarrime.github.io/tic-tac-toe/)


## Features

- Two-player mode
- Custom player names
- Turn indicator
- Win detection
- Draw detection
- Restart anytime
- Dynamic interface
- Responsive design

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/tic-tac-toe.git
```

2. Navigate to the project 
```sh
cd tic-tac-toe
```
3. Open index.html in your browser


## How to use

1. Open the website in your browser
2. View the game board
3. Start playing:
   - Player X goes first, then Player O
   - Click on any empty cell to place your mark
4. Update player names:
   - Click on the chnage names button
   - Type in your custom names
5. Game outcomes:
   - Get three in a row to win
   - The board fills up with no winner = draw
6. Manage the game:
   - Click "Restart" to restart the board
